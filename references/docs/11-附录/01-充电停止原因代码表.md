# 充电停止原因代码表

## 充电完成 (0x40-0x49)

| 代码 | 具体原因 |
|------|----------|
| 0x40 | 结束充电，APP远程停止 |
| 0x41 | 结束充电，SOC达到100% |
| 0x42 | 结束充电，充电电量满足设定条件 |
| 0x43 | 结束充电，充电金额满足设定条件 |
| 0x44 | 结束充电，充电时间满足设定条件 |
| 0x45 | 结束充电，手动停止充电 |
| 0x46~0x49 | 其他方式(预留) |

## 使用说明

该代码表用于交易记录(0x3B)中的"停止原因"字段,标识充电结束的具体原因。根据代码可以判断:
- 是否正常结束
- 启动阶段失败原因
- 充电过程异常原因
