# 通信协议结构

本协议的通信协议结构如图1所示。本协议采用的TCP/IP传输定义与标准定义一致。

| 应用功能 | 初始化 | 用户进程 |
|----------|--------|----------|
| 本协议中定义的ASDU | | 应用层(第7层) |
| APCI(应用规约控制信息)传输接口(用户到TCP的接口) | | 应用层(第7层) |
| TCP/IP 协议子集 | | 传输层(第4层) |
| TCP/IP 协议子集 | | 网络层(第3层) |
| TCP/IP 协议子集 | | 链路层(第2层) |
| TCP/IP 协议子集 | | 物理层(第1层) |
