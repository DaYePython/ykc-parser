# 数据格式定义

## 数据格式类型

数据格式包括:
- **BCD码**: 二进制编码的十进制
- **BIN码**: 二进制码,均为低位在前高位在后
- **ASCII码**: 美国信息交换标准代码

## 小数处理规则

协议中小数值均乘倍率(保留小数点位数)上送平台。

**示例**: 电压为225.1V,保留一位小数,上送到平台值为2251,即0x8CB

## CP56Time2a时间格式

| 字节 | 内容 | 说明 |
|------|------|------|
| Byte1 | Milseconds(D7-D0) | 毫秒低字节 |
| Byte2 | Milseconds(D15-D8) | 毫秒高字节 |
| Byte3 | IV(D7) RES1 Minutes(D5-D0) | 分钟 |
| Byte4 | SU(D7) RES2 Hours(D4-D0) | 小时 |
| Byte5 | DAY of WEEK DAY of MONTH(D4-D0) | 星期和日期 |
| Byte6 | RES3 Month(D3-D0) | 月份 |
| Byte7 | RES4 Years(D6-D0) | 年份 |
