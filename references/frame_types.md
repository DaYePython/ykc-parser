# 云快充协议帧类型定义

## 帧类型分类

- 充电桩定义的帧类型码为**奇数**
- 运营平台定义的帧类型码为**偶数**

## 注册心跳类 (0x01-0x0A)

| 帧类型码 | 名称 | 方向 | 说明 |
|---------|------|------|------|
| 0x01 | 充电桩登录认证 | 桩→平台 | 桩每次复位或通信离线都需重新登录 |
| 0x02 | 登录认证应答 | 平台→桩 | |
| 0x03 | 充电桩心跳包 | 桩→平台 | |
| 0x04 | 心跳包应答 | 平台→桩 | |
| 0x05 | 计费模型验证请求 | 桩→平台 | |
| 0x06 | 计费模型验证应答 | 平台→桩 | |
| 0x09 | 充电桩计费模型请求 | 桩→平台 | |
| 0x0A | 计费模型请求应答 | 平台→桩 | |

## 实时数据类 (0x12-0x25)

| 帧类型码 | 名称 | 方向 | 说明 |
|---------|------|------|------|
| 0x12 | 读取实时监测数据 | 平台→桩 | 平台主动请求 |
| 0x13 | 上传实时监测数据 | 桩→平台 | 周期/变位/召唤上送 |
| 0x15 | 充电握手 | 桩→平台 | |
| 0x17 | 参数配置 | 桩→平台 | |
| 0x19 | 充电结束 | 桩→平台 | |
| 0x1B | 错误报文 | 桩→平台 | |
| 0x1D | 充电阶段BMS中止 | 桩→平台 | |
| 0x21 | 充电阶段充电机中止 | 桩→平台 | |
| 0x23 | BMS需求/充电机输出 | 桩→平台 | |
| 0x25 | BMS信息 | 桩→平台 | |
